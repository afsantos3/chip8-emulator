# CHIP-8 Emulator
A CHIP-8 emulator in C++

## Compiling and Running
Requires cmake:

Linux:
```
$ sudo apt-get install cmake
```

macOs:
```
$ brew install cmake
```

Compile:
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

Run:
```
./Chip8Emulator ../path/to/rom/<ROM file>
```
